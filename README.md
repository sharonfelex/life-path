# LifePath

Prototype assets for the LifePath ambulance emergency response platform.

## Contents
- `flutter_app/`: Flutter shell app with emergency mode, GPS polling, and MQTT streaming.
- `ml/`: Python Random Forest training script for Bengaluru traffic delay prediction.
- `backend/`: .NET 8 Web API with SignalR, EF Core models, and proximity logic.
- `iot/`: ESP32 + SIM800L MQTT telemetry simulator.
